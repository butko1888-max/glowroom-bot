# 💅 Beauty Bot — Телеграм бот для записи на бьюти услуги

## Что умеет бот

**Для клиентов:**
- 📅 Запись на услугу (выбор услуги → даты → времени → подтверждение)
- 💰 Просмотр прайс-листа
- 📋 Просмотр своих записей
- 🔔 Автоматическое напоминание за день до визита

**Для мастера (админ-панель):**
- Записи на сегодня
- Все будущие записи
- База клиентов
- 📢 Рассылка сообщений всем клиентам

---

## Шаг 1 — Создай бота в Telegram

1. Открой @BotFather в Telegram
2. Напиши `/newbot`
3. Придумай имя и username для бота
4. Скопируй **токен** — он выглядит так: `7123456789:AAF...`

---

## Шаг 2 — Узнай свой Telegram ID

1. Напиши боту @userinfobot
2. Он пришлёт твой `Id` — это число, например `123456789`
3. Это будет ADMIN_ID — только ты увидишь админ-панель

---

## Шаг 3 — Настрой бота

Открой файл `bot.py` и измени данные мастера в функции `show_contacts`:
```python
"👩 Мастер: Имя мастера\n"
"📱 Телефон: +380 XX XXX XX XX\n"
"📍 Адрес: Укажи адрес студии\n"
```

Также можешь изменить список услуг и цены в словаре `SERVICES`:
```python
SERVICES = {
    "lam_brows": {"name": "💆 Ламинирование бровей", "price": 600, "duration": 60},
    ...
}
```

И доступное время для записи:
```python
AVAILABLE_TIMES = ["10:00", "11:00", ..., "19:00"]
```

---

## Шаг 4 — Задеплой на Railway (бесплатно)

1. Зарегистрируйся на [railway.app](https://railway.app) через GitHub
2. Создай новый проект → **Deploy from GitHub repo**
   - Загрузи все файлы в GitHub репозиторий (или используй Railway CLI)
3. В настройках проекта найди **Variables** и добавь:
   - `BOT_TOKEN` = твой токен от BotFather
   - `ADMIN_ID` = твой Telegram ID
4. Нажми **Deploy** — бот запустится автоматически!

### Альтернатива — запуск локально (для теста):

```bash
# Установи зависимости
pip install -r requirements.txt

# Запусти с переменными окружения
BOT_TOKEN="твой_токен" ADMIN_ID="твой_id" python bot.py
```

## Как пользоваться админ-панелью

1. Напиши боту команду `/admin`
2. Доступно только с твоего аккаунта (ADMIN_ID)
3. Выбирай нужный раздел кнопками

### Рассылка:
1. `/admin` → 📢 Сделать рассылку
2. Напиши текст — он уйдёт всем клиентам кто когда-либо писал боту

## Часто задаваемые вопросы

**Где хранятся данные?**
В файле `beauty_bot.db` на сервере Railway. При перезапуске данные сохраняются.

**Можно добавить новые услуги?**
Да — просто добавь новую строку в словарь `SERVICES` в файле `bot.py`.

**Как изменить рабочие часы?**
Измени список `AVAILABLE_TIMES` в `bot.py`.

**Как отменить запись клиента?**
Пока через базу данных напрямую. В следующей версии можно добавить кнопку отмены.
